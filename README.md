# 神秘嵌入式流体模拟器
受流体模拟项链项目的启发，用ch32v303（或者203？）复刻的跑在单片机上的流体模拟器。参考代码在reference.html中（复制于流体模拟项链的开源项目，来源详见文件内容）（本项目有大量codex成分，小心AI Slop）

## 目前功能
已经写好了物理模拟部分的代码，测试正常。
在CH32V303上128个粒子能跑到120Hz，无需浮点。
